# Omid YouTube FA Subtitle v1.1.0

Chrome Extension برای استخراج زیرنویس ویدیوهای YouTube، ساخت فایل JSON قابل ترجمه، بارگذاری ترجمه فارسی، و نمایش زیرنویس فارسی به‌صورت overlay روی ویدیو.

این افزونه ترجمه را خودش انجام نمی‌دهد. کار اصلی آن این است که زیرنویس موجود ویدیو را با تایم‌کد دقیق خارج کند، تو فایل را به مدل زبانی بدهی، مدل فقط فیلد `fa` را پر کند، و بعد همان JSON ترجمه‌شده دوباره داخل افزونه بارگذاری شود.

---

## قابلیت‌ها

- استخراج ترک‌های زیرنویس موجود از YouTube
- انتخاب ترک manual یا auto-generated
- ساخت JSON آماده برای ترجمه
- ساخت فایل TXT شامل پرامپت آماده
- بارگذاری JSON ترجمه‌شده
- نمایش زیرنویس فارسی راست‌به‌چپ روی ویدیو
- تنظیم اندازه فونت و فاصله زیرنویس از پایین ویدیو
- ذخیره ترجمه برای هر ویدیو داخل مرورگر
- بدون بک‌اند، بدون API key و بدون سرور خارجی

## نصب در Chrome

1. ریپو را دانلود یا clone کن.
2. Chrome را باز کن.
3. برو به:

```text
chrome://extensions
```

4. گزینه `Developer mode` را روشن کن.
5. روی `Load unpacked` بزن.
6. پوشه همین پروژه را انتخاب کن.
7. یک ویدیوی YouTube باز کن و صفحه را یک بار کامل refresh کن.
8. روی آیکن افزونه بزن.

---

## نحوه استفاده

### 1. انتخاب ویدیو

یک ویدیوی YouTube باز کن که subtitle یا caption داشته باشد.

### 2. انتخاب ترک زیرنویس

در popup افزونه، ترک موردنظر را انتخاب کن. اگر ترک manual وجود داشت، معمولاً بهتر از auto-generated است.

### 3. دانلود JSON برای ترجمه

روی دکمه دانلود JSON بزن. افزونه زیرنویس را می‌خواند و یک فایل JSON با زمان شروع، زمان پایان، متن اصلی و فیلد خالی فارسی می‌سازد.

### 4. ترجمه با مدل زبانی

فایل JSON را به مدل بده و از مدل بخواه فقط فیلد `fa` را پر کند.

### 5. بارگذاری ترجمه فارسی

بعد از ترجمه:

1. خروجی مدل را با فرمت `.json` ذخیره کن.
2. در popup افزونه فایل را انتخاب کن.
3. روی بارگذاری زیرنویس فارسی بزن.
4. زیرنویس فارسی روی ویدیو نمایش داده می‌شود.

## نمونه JSON خروجی

```json
{
  "schema": "omid-youtube-fa-subtitle-v1.1",
  "video": {
    "id": "VIDEO_ID",
    "title": "Video title",
    "url": "https://www.youtube.com/watch?v=VIDEO_ID"
  },
  "cues": [
    {
      "id": 1,
      "start": 0.5,
      "end": 3.2,
      "start_time": "00:00:00.500",
      "end_time": "00:00:03.200",
      "source": "Original subtitle text",
      "fa": ""
    }
  ]
}
```

## پرامپت پیشنهادی برای ترجمه

```text
این فایل JSON زیرنویس زمان‌دار YouTube است.

وظیفه تو:
برای هر آیتم داخل آرایه cues فقط فیلد fa را با ترجمه فارسی روان، طبیعی و مناسب زیرنویس پر کن.

قوانین:
1. خروجی فقط JSON معتبر باشد.
2. هیچ توضیحی بیرون JSON ننویس.
3. schema، video و cues را حذف نکن.
4. مقدارهای id، start، end، start_time، end_time و source را تغییر نده.
5. فقط fa را پر کن.
6. ترجمه کوتاه، روان و مناسب نمایش روی ویدیو باشد.
7. اگر جمله ناقص بود، با توجه به قبل و بعد ترجمه قابل فهم بده اما زمان‌بندی را تغییر نده.
8. اگر cue فقط موسیقی، صدا یا افکت بود، ترجمه کوتاه مناسب بنویس مثل [موسیقی].

بعد از ترجمه، همان JSON کامل را برگردان.
```

## رفع خطاهای رایج

اگر زیرنویس پیدا نشد یا خروجی خالی بود:

1. صفحه YouTube را refresh کن.
2. زیرنویس خود YouTube را یک بار دستی روشن کن.
3. چند ثانیه صبر کن.
4. دوباره از افزونه خروجی بگیر.

اگر باز هم مشکل بود:

- ترک دیگری را انتخاب کن.
- با ویدیوی دیگری تست کن.
- مطمئن شو ویدیو واقعاً caption دارد.
- افزونه را از `chrome://extensions` یک بار reload کن.

---

## محدودیت‌ها

- فقط روی ویدیوهایی کار می‌کند که YouTube برای آن‌ها caption قابل دسترسی داشته باشد.
- ترجمه داخل افزونه انجام نمی‌شود.
- اگر مدل ساختار JSON را خراب کند، فایل ترجمه‌شده بارگذاری نمی‌شود.
- اگر YouTube ساختار داخلی Player را تغییر دهد، ممکن است افزونه نیاز به آپدیت داشته باشد.
